A linguagem HTML

HTML (abreviação para a exprssão ingelsa HyperText Marckup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.


Alguns recursos da linguagem HTML vistos até este momento:

Recursos Tags Associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenados
<ol> e <li> Criar listas de itens não ordenados
Imagens <img> Inserir imagens
