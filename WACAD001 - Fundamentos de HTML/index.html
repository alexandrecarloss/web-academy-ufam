História do HTML

Tim Berners-Lee (físico britânico) criou o HTML em 1991, enquanto trabalhava no CERN (Organização Europeia para Pesquisa Nuclear). Ele desenvolveu o HTML como parte de um projeto para facilitar a troca de informações entre pesquisadores usando a internet. O HTML permitiu que os documentos fossem formatados e vinculados uns aos outros, dando origem à World Wide Web como a conhecemos hoje.
